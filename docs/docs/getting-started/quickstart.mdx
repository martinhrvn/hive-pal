# Quick Start Wizard

Welcome to Hive-Pal! This interactive guide will help you get up and running in minutes. Follow these steps to set up your beekeeping management system.

## Prerequisites

Before you begin, ensure you have:
- Docker and Docker Compose installed (for Docker setup)
- Node.js 18+ and PostgreSQL (for manual setup)
- A modern web browser

## Step 1: Choose Your Installation Method

<Tabs>
  <TabItem value="docker" label="Docker (Recommended)" default>

### Docker Installation

The fastest way to get started with Hive-Pal.

```bash
# Clone the repository
git clone https://github.com/martinhrvn/hive-pal.git
cd hive-pal

# Copy environment files
cp apps/backend/.env.example apps/backend/.env
cp apps/frontend/.env.example apps/frontend/.env

# Start the application
docker-compose up -d
```

:::tip
Docker Compose will automatically:
- Set up PostgreSQL database
- Run database migrations
- Start the backend API
- Start the frontend application
:::

  </TabItem>
  <TabItem value="manual" label="Manual Installation">

### Manual Installation

For advanced users who want more control over the setup.

```bash
# Clone the repository
git clone https://github.com/martinhrvn/hive-pal.git
cd hive-pal

# Install dependencies
pnpm install

# Set up environment files
cp apps/backend/.env.example apps/backend/.env
cp apps/frontend/.env.example apps/frontend/.env

# Configure your database in apps/backend/.env
# DATABASE_URL="postgresql://user:password@localhost:5432/hivepal"

# Run database migrations
cd apps/backend
pnpm prisma migrate dev

# Start the application
cd ../..
pnpm dev
```

  </TabItem>
</Tabs>

## Step 2: Access the Application

Once the installation is complete:

1. Open your browser and navigate to `http://localhost:5173`
2. You should see the Hive-Pal landing page

:::note
- Frontend runs on port **5173**
- Backend API runs on port **3000**
- API documentation available at `http://localhost:3000/api-docs`
:::

## Step 4: Initial Setup Wizard

After logging in for the first time, you'll be guided through the initial setup:

### 4.1 Create Your First Apiary

<div style={{textAlign: 'center', margin: '2rem 0'}}>
  <div style={{padding: '1rem', background: '#f0f0f0', borderRadius: '8px'}}>
    [Video: Creating Your First Apiary - Coming Soon]
  </div>
</div>

1. Click the **"+"** button (FAB) in the bottom right
2. Select **"New Apiary"**
3. Fill in the apiary details:
   - **Name**: Give your apiary a memorable name
   - **Location**: Enter the address or coordinates
   - **Notes**: Add any relevant information about the location
4. Click **"Save"**

### 4.2 Add Your First Hive

<div style={{textAlign: 'center', margin: '2rem 0'}}>
  <div style={{padding: '1rem', background: '#f0f0f0', borderRadius: '8px'}}>
    [Video: Adding Your First Hive - Coming Soon]
  </div>
</div>

1. Navigate to your apiary
2. Click the **"+"** button
3. Select **"New Hive"**
4. Enter hive information:
   - **Name/Number**: Unique identifier for the hive
   - **Type**: Select hive type (Langstroth, Top Bar, etc.)
   - **Number of boxes**: Current configuration
   - **Status**: Active, Inactive, etc.
5. Click **"Save"**

### 4.3 Record Your First Inspection

1. Select a hive from your apiary
2. Click **"New Inspection"**
3. Fill in the inspection details:
   - Date and time
   - Weather conditions
   - Colony strength
   - Queen status
   - Observations
4. Click **"Save Inspection"**

## Step 5: Explore Key Features

### Quick Actions Menu (FAB)

The floating action button (FAB) in the bottom right provides quick access to:
- Create new apiaries
- Add hives
- Record inspections
- Schedule tasks
- Add harvest records

### Dashboard Overview

Your dashboard displays:
- Recent inspections
- Upcoming scheduled tasks
- Weather forecast for your apiaries
- Quick statistics

### Mobile Usage

Hive-Pal is fully responsive and works great on mobile devices:
- Record inspections in the field
- Quick access to hive history
- Offline capability (data syncs when reconnected)
