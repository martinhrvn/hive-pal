# Quick Check

A **Quick Check** records a lightweight observation about a hive without the detail of a full [inspection](./inspections). Use it for fast walk-bys — "calm and active", "needs a super", "spotted varroa" — when you don't need ratings or scoring.

## When to Use It

- You're walking the apiary and want to jot a quick status.
- You notice something worth remembering (a reminder) but aren't doing a full inspection.
- You want to attach a photo or two to the hive's timeline.

## Recording a Quick Check

1. Open a hive's detail page and choose **Quick Check** from the side menu (or the quick-action button).
2. Tap one or more **Observation** tags to describe what you see:
   - *Calm, Defensive, Aggressive, Nervous*
   - *Varroa Present, Small Hive Beetle, Wax Moths, Ants Present*
   - *Healthy, Active, Sluggish, Thriving*
3. Optionally tap **Reminder** tags for follow-up items:
   - *Honey Bound, Overcrowded, Needs Super, Queen Issues, Requires Treatment, Low Stores, Prepare for Winter*
4. Add an optional **Note** (up to 2,000 characters).
5. Attach up to **5 photos** if file storage is enabled.
6. Tap **Save Quick Check**.

You need at least one tag or a note to save. The check is timestamped and appears on the hive's timeline alongside inspections.

## Quick Check vs Inspection

| | Quick Check | Inspection |
|---|---|---|
| Effort | Seconds | A few minutes |
| Ratings / scoring | No | Yes |
| Actions (feeding, treatment) | No | Yes |
| Photos | Yes (up to 5) | Yes |
| Best for | Field notes, reminders | Full colony assessment |

## FAQ

**Does a quick check affect hive scores?**
No. Quick checks are observational notes only — they don't feed the inspection scoring system. Use a full [inspection](./inspections) when you want scores and trends.

**Can I add photos?**
Yes, up to five per quick check, when your installation has file storage enabled.

**Where do quick checks show up?**
On the hive's timeline, interleaved with inspections and other events.
